# sdd-tutorial-todo-app

---

## 開発者が行うこと

### ディレクトリ作成
```
/sdd-tutorial-todo-app
  + /generated  # AI生成のコードを管理
  + /packages
    + /frontend
    + /backend
    + /shared
```
* フレームワークインストールも併せて行う

### モノレポ設定
* npm workspace 設定
* tsconfig 設定
* tsconfig はベースconfigを利用せず、各パッケージから明示的に依存を指定する

### sharedスキーマのコーディング
* 通信用のZodSchemaなど、中核となるスキーマを定義する

### コードレビュー
* AIが生成したコードに対してレビューを行う。

---

## AIが行うこと

### UI構成
* react + tailwindcss を使ったUI整形
* 許可されたUIのみ編集する
* UI編集はUIの見た目・細部動作もに編集可能

### ロジックミスの指摘
* 細部のロジックミスを指摘する
* ただし、ロジックコードの書き換えは100行未満とする

### 型に従ったコーディング
* 原則asとanyによるキャスティングを許可しない
* する必要のある部分は必ず理由を添えること

### 改修
* 開発者が改修を求めた場合は改修する
* 開発者が改修を求めなかった場合は開発者に委ねる

### ボイラーテンプレートの自動化
* 開発者がボイラーテンプレート（単純機能の追加）などを求めた場合、既存のコードに従って機能追加のみ行う

### 指示に沿った行動
* 1指示あたり1タスクのみ行い、複数の種類のタスクを一度に行うことを避ける

### 出力
* コード出力はgeneratedディレクトリに出力する
* ディレクトリへの出力ができない場合はチャット上に出力する
